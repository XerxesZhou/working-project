什么是合同？

合同是用来管理所有已经发生的销售记录，合同、回款是用于记录合同信息，制定回款计划，记录回款情况，记录开票情况。

注：1、合同分类、付款方式和合同状态所有数据系统都设置了默认值，若修改相关默认值，建议在使用系统前修改；

2、并且所有数据是无法删除的，如需调整，点击导航栏系统管理中数据字典，在展开的页面上点击合同，在展开的页面上可以通过字段左侧的开关按钮来设定是否启用。

新增合同

点击右上方“新增合同”按钮，按照提示填写即可，其中红色星标为必填项。注：可以在系统后台设置“自定义字段设置”内更改必填项目。

删除合同

点击导航栏左侧“合同”，在展开的页面中点击“删除所选”

合同信息管理

点击合同池中，任何一个合同可进入合同详情页面。合同信息的详细页面，可直接针对该合同新增回款、添加产品、转移给其他人等。在右侧可直接新增跟进记录。

合同状态

随时了解合同情况

编辑合同信息

需要修改或者完善合同信息，点击商机二级页面中“编辑”，进入编辑合同页面，在展开的页面中完善或修改信息，若想要信息更加完整，点击“展开更多信息”即可

导入合同

通过此按钮可批量导入合同，下载导入数据模板，按照模板中的项目整理要导入的商机数据。注：1）一次性可导入4000条。2 ) 数据模板中的表头信息，是来自于系统设置中配置的商机字段，表头名称不能更改，表头行不能删除。
