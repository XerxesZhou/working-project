什么是联系人？

联系人是指所属于某个客户的老板、经理或者其他类型的人。

注：1、联系人类别所有数据系统都设置了默认值，若修改相关默认值，建议在使用系统前修改；

2、所有数据是无法删除的，如需调整，点击导航栏系统管理中数据字典，在展开的页面上点击线索，在展开的页面上可以通过字段左侧的开关按钮来设定是否启用。

导入联系人

通过此按钮可批量导入联系人，下载导入数据模板，按照模板中的项目整理要导入的联系人数

删除联系人

首先点击联系人页面中姓名前面的小方块，然后点击按钮“删除所选”，确定之后在删除页面上点击”确认“即可

联系人信息管理

点击联系人池中，任何一个联系人可进入详情页面。联系人信息的详细页面，可以直接查看该联系人相关信息，包括：基本信息、商机等。

编辑联系人信息

修改或者完善联系人信息，点击联系人二级页面中“编辑”，即可进入到编辑联系人页面，在展开的页面上修改和完善信息，若想要填写信息更加完整，点击“展开更多信息”即可

新增联系人

点击右上方“新增联系人”按钮，按照提示填写即可，其中红色星标为必填项。注：可以在系统后台设置“自定义字段设置”内更改必填项目。
