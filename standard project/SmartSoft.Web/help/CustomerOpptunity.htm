什么是商机？

商机指的是客户的某个具体的购买需求，比如，有明确的购买意向（具体某个产品）、大致的预算、大致的采购时间和明确的客户采购人。

注：1、商机来源、所处阶段和商机类别所有数据系统都设置了默认值，若修改相关默认值，建议在使用系统前修改；

2、所有数据是无法删除的，如需调整，点击导航栏系统管理中数据字典，在展开的页面上点击商机，在展开的页面上可以通过字段左侧的开关按钮来设定是否启用。

新增商机

点击右上方“新增商机”按钮，按照提示填写即可，其中红色星标为必填项。注：可以在系统后台设置“自定义字段设置”内更改必填项目。

删除客户

点击导航栏左侧“商机”，在展开的页面中点击“删除所选”

商机信息管理

点击商机池中，任何一个商机可进入详情页面。商机信息的详细页面，可以直接针对该商机添加产品、转移给他人、新建合同、关联产品、增加附件等。在右侧可直接新增跟进记录。

商机状态

随时了解客户跟进情况，知道预计销售

商机转成合同

赢单的商机转为合同，点击商机二级页面中“转成合同”，即可进入新增合同页面，在展开的页面上填写信息

编辑商机信息

需要修改或者完善商机信息，点击商机二级页面中“编辑”，进入编辑商机页面，在展开的页面中完善或修改信息，若想要信息更加完整，点击“展开更多信息”即可

导入商机

通过此按钮可批量导入商机，下载导入数据模板，按照模板中的项目整理要导入的商机数据。注：1）一次性可导入4000条。2 ) 数据模板中的表头信息，是来自于系统设置中配置的商机字段，表头名称不能更改，表头行不能删除。
